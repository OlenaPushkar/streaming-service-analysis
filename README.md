# Streaming Service User Behavior Analysis

## Project Goal

Analyze user viewing behavior on a streaming platform and identify factors affecting engagement and churn.

## Tools

- Google Sheets
- SQL
- Data Cleaning
- KPI Analysis
- Dashboarding

## Dataset

The dataset contains information about:

- Users
- Sessions
- Content genres
- Watch time
- Subscription plans
- Churn status

## Key KPIs

- Total Watch Time: 56,532 minutes
- Average Watch Time per Session: 94.2 minutes
- Completion Rate: 71%
- Highest Churn Rate: 10.7%

## Main Findings

- Thriller and Action are the most popular genres.
- User activity increases month over month.
- Basic subscription has the highest churn rate.
- Standard subscription has the lowest churn rate.

## Recommendations

- Invest more in Thriller and Action content.
- Improve Basic subscription offering.
- Continue monitoring churn trends.
